Your First Python Program

  • Due Monday by 11:59pm
  • Points 25
  • Submitting a text entry box or a file upload

Purpose

To become familiar with Visual Studio Code and GitHub by completing a simple task.

Task Overview

There are two ways you can complete your programming projects:

  1. Work locally in Visual Studio Code and upload your changes via file upload.
  2. Connect Visual Studio Code to your remote repository and Commit and Push changes directly.

Directions

Ready for your first Python program?! Let's use the tried and true Hello World program as our first program. All you need to do is enter the following (after filling in your name and date in the comments):

# This is my first Python program.
print("Hello World!")

1. Before completing this assignment, you will need to download and unzip the assignment files and instructions from https://github.com/Western-Wyoming-Computer-Science/COSC1010 Links to an external site.. You should save them locally (to a COSC1010 folder on your computer and/or storage device). 

2. Optional: upload the files to your GitHub COSC1010 Repository by file upload (you could also clone the files when you create your new repository...there are multiple ways of doing this and you will figure out which methods work better for you). Upload the files by clicking the Add File > Upload iles option. Copy the contents of the unzipped files to your repository and click the Commit Changes button.

3. Open Visual Studio Code and open your COSC1010 local folder (or connect remotely to your GitHub COSC1010 Repository). To open your local folder, go to File > Open Folder and navigate to the COSC1010 folder. Highlight (single-click, do not double-click) and click the Select Folder button.

4. Navigate to the Your First Python Program folder and click on the instructions.md file. Every programming project will contain an instructions.md and main.py file (some may even contain additional .txt and .py supporting files).

5. Open the main.py file and complete the assignment. When you are done, be sure to run your code by clicking on the icon at the top right (always test your code and fix any errors before submitting). Your code and output should look very similar to the screenshot below.

6. Save your work in Visual Studio Code (Ctrl + S for Windows users or File > Save All), then upload your completed files to your GitHub COSC1010 repository. Click the Commit Changes button when done.

7. Verify that the changes uploaded by navigating to the Your First Python Program folder (in GitHub) and clicking on the main.py. If you see your code, you are good!

8. Submit your GitHub COSC1010 URL when you are ready for me to grade this assignment.

Ready! Set! Go!!
